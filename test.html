IT之家 3 月 28 日消息，苹果今日向开发者推送了 macOS Sequoia 15.4 的第二个 RC 候选版，而首个候选版本发布于 3 月 25 日。

  • 15.4 RC (24E246)

  • 15.4 RC2 (24E247)

开发者用户可通过系统设置的软件更新版块获取 macOS 测试版，需使用与开发者账户关联的 Apple ID 登录。本次更新的核心功能包括：

  • macOS Sequoia 15.4 首次为 Mac 引入邮件分类功能，将邮件应用重新组织为交易、更新、推广和主要邮件等类别，优先显示重要邮件。

  • 为 Apple News+ 订阅用户创建“美食专区”，收录各种食谱与餐饮指南等相关内容。

  • Image Playground 新增素描风格特效，为用户提供更多创意选择。

  • 照片应用支持创建记忆短片功能，帮助用户更好地整理回忆。

  • 苹果智能（Apple Intelligence）扩展新的语言，提升了智能体验。

官方路线图显示，苹果 macOS Sequoia 15.4 正式版计划于 4 月初发布。若无重大技术问题，当前的 RC 候选发布版将直接作为最终版本发布。

如何升级 iOS / iPadOS / watchOS / macOS 开发版和公测版？

・ 公开测试版：需注册 Apple Beta 版软件计划，之后通过【设置】【通用】【软件更新】【Beta 版更新】来升级；

・ 开发预览版：需登录注册苹果开发者计划，之后通过【设置】【通用】【软件更新】来升级。
