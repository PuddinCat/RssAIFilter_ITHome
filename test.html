IT之家 4 月 8 日消息，标致昨日在国内发布第四代 DJANGO 系列车型，共分为四个版本的车型。其中，全新 DJANGO LIGHT 指导价 15800 元，全新 DJANGO CLASSIC 指导价 16800 元，全新 DJANGO TT 指导价 17800 元，全新 DJANGO CAFERACER 指导价 17800 元。

据介绍，全新 DJANGO LIGHT 将提供闪光银、柠檬黄 / 魅影灰、爱琴海蓝、樱桃红 / 冰雪白四种配色，拥有双通道 ABS、腕表 TFT 互联仪表、智能无钥匙启动、全车 LED 灯光系统等配置。

全新 DJANGO CLASSIC 提供星月白 / 碳晶黑 / 奇异棕、冰雪白 / 碳晶黑配色，尺寸为 1925×710×1200mm、轴距为 1350mm、座高 780mm。新车配备双通道 ABS、米其林轮胎、腕表 TFT 互联仪表、智能无钥匙启动、全车 LED 灯光系统和飞旋后搁脚等配置。

全新 DJANGO TT 提供海岸白 / 秋叶黄、海岸白 / 仙踪绿两种配色，该车配备文件夹与复古包、狮跑前泥板、多功能后货架、全地形轮胎等，拥有双通道 ABS、腕表 TFT 互联仪表、智能无钥匙启动、全车 LED 灯光系统以及飞旋后搁脚等。

全新 DJANGO CAFERACER 提供冰雪白、闪光银、V 黑三种配色，搭载全新第二代 Easymotion 150cc 四气门高效水冷发动机、双通道 ABS、米其林轮胎、腕表 TFT 互联仪表、智能无钥匙启动、全车 LED 灯光系统、飞旋后搁脚、狮耳整流罩、高耸驼峰座壳罩、圆形手把后视镜 / 绅士竖条把套、锋锐前泥板、银翼扶手、赛车徽章。

IT之家从标致摩托获悉，全新 DJANGO 升级全新腕表 TFT 互联仪表，可以显示行车信息、主题模式、显示模式、链接设置、时钟调节功能。车机具备互联功能，通过 App 可查看电压、保养里程、平均油耗等，还可实现投屏导航。

新车搭载 150cc 四气门高效水冷发动机，最大功率 10.8kW、最大扭矩 13.8 N・m、最大功率及最大扭矩提前 1000rpm。官方表示，新车最高车速提升 8%、噪音降低约 10%、百公里油耗降低 5%。
