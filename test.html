IT之家 4 月 19 日消息，科技媒体 Windows Report 昨日（4 月 18 日）发布博文，报道称微软计划在安卓版 Microsoft Edge 浏览器中，弃用 Mobile Wallet，目前用户已无法访问相关页面。

桌面版 Edge 浏览器支持 Wallet 功能，能安全存储支付方式、密码等信息，并允许用户为密码添加备注。不仅如此，Wallet 还支持查看 Microsoft Rewards 或 Cashback 余额，并提供虚拟卡选项，进一步保障在线购物安全。

遗憾的是，安卓版 Edge 浏览器迟迟没有上线 Mobile Wallet，近期发现的一个实验性 Flag 更显示微软计划弃用该功能。

这意味着安卓用户可能永远无法体验这一便利功能。Edge 安卓版的 edge://wallet 页面目前也无法访问，显示页面存在问题或已永久迁移。微软尚未正式宣布安卓版 Wallet 计划，这让用户的期待落空。
